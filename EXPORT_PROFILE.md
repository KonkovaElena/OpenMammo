# Export Profile: Mammography Standalone v1

Date: 2026-04-05
Status: active

## Mission

Export a standalone clinician-in-the-loop mammography second-opinion product that starts with a narrow FFDM-only workflow kernel and grows into a full review and delivery workflow without inheriting unrelated platform mass.

## Product Truths

1. The standalone boots independently.
2. The standalone declares its mission, scope, and non-goals in machine-readable form.
3. The standalone exposes health, readiness, and metrics behavior.
4. The standalone preserves request correlation headers.

## Mandatory Kernel

1. entry chain
2. bootstrap composition root
3. configuration loading
4. metrics facade
5. health and readiness behavior
6. request correlation behavior
7. authority docs for mission, evidence, and roadmap

## Explicit Non-Goals

1. DBT in v1
2. ultrasound
3. breast MRI
4. autonomous diagnosis
5. PACS replacement
6. model training platform

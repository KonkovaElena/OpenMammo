# Code of Conduct

This repository expects respectful, technically precise, and evidence-based collaboration.

## Expected Behavior

- discuss ideas and code directly, without personal attacks
- keep claims grounded in code, tests, or cited evidence
- treat medical and safety topics with appropriate care and precision
- assume good intent while staying rigorous about correctness

## Unacceptable Behavior

- harassment, abuse, or discriminatory language
- publishing private or sensitive information without permission
- inflammatory or intentionally misleading technical claims
- derailing review threads with unrelated disputes

## Enforcement

Maintainers may edit, hide, or remove content that violates this code of conduct and may restrict repository participation for repeated or severe violations.

For private concerns, use the security or maintainer contact path described in [SECURITY.md](SECURITY.md).
